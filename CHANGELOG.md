# Change Log

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## [6.0.0] - 2025-01-20

### Changed

- Required Node >=22
- Splitted the "Memoize" decorator into 2: "MemoizeMethod" and "MemoizeGetter"

## [5.0.0] - 2023-05-30

### Changed

- Required Node >=18
- Optimized for getter and method without hashFunction

## [4.0.0] - 2022-11-24

### Changed

- Renamed the package into @prismamedia/memoize

## [3.0.1] - 2022-09-09

### Changed

- Upgraded dev-dependencies

## [3.0.0] - 2022-05-30

### Changed

- Published as ESM

## [2.1.0] - 2022-05-19

### Changed

- Added the ability to not memoize a result

## [2.0.0] - 2022-01-20

### Changed
